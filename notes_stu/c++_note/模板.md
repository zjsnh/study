# 模板

泛型编程

## 函数

```c++
template<typedename T1,.....>
```

```c++
template<typename T>
void swap(T& a, T& b)
{
    T temp = a;
    a = b;
    b = temp;
}

int main()
{
    int a = 5,b = 6;
    swap(a, b);

   std::cout << a << " " << b;


    return 0;
}
```

但是，这两个参数不一样的话，是会报错的，这时候我们需要对变量进行

```c++
int a = 10;
double b = 20.0;
```

强制转换

```c++
Add(a, (int)d);
```

或者显式实例化

```c++
Add<int>(a, b);
```
